# IDEAS — רעיונות ופערים שהסוכנים מציעים

> כשסוכן מזהה רעיון לשיפור או פער במוצר — אבל הוא לא ב-MASTER_PLAN ולא ברור שהוא
> מספיק חשוב — הוא רושם אותו כאן במקום לבנות אותו על דעת עצמו.
> את (כרמל / תמר) מאשרת או דוחה. רעיון מאושר → את מעבירה ל-INBOX או ל-MASTER_PLAN.

פורמט:
```
### [ממתין] YYYY-MM-DD — כותרת הרעיון
**הפער/הרעיון:** ...
**ערך למשתמש:** ...
**מאמץ משוער:** קטן / בינוני / גדול
```
סטטוסים: `[ממתין]` → `[מאושר]` / `[נדחה]`
