# INBOX — תיבת הודעות לסוכנים

> כאן את (תמר / כרמל) כותבת הוראות לצוות הסוכנים האוטונומי.
> כל הרצה קוראת את הקובץ הזה **ראשון**. הודעות חדשות גוברות על ה-MASTER_PLAN לאותה הרצה.
> הסוכן עונה תחת ההודעה ומסמן אותה `[DONE]`. הודעות `[DONE]` נשארות כתיעוד.

---

## איך משתמשים
1. כתבי הודעה חדשה למטה תחת `## הודעות פתוחות`, בפורמט:
   ```
   ### [פתוח] 2026-06-24 — נושא קצר
   הטקסט שלך בעברית. מה את רוצה שיקרה.
   ```
2. שמרי את הקובץ. ההרצה הבאה תקרא אותה.
3. הסוכן יבצע, יענה מתחת, וישנה `[פתוח]` ל-`[DONE]`.
