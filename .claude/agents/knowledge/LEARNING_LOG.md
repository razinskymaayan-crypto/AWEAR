# Learning Log — daily retrospectives

Once a day the retrospective facilitator turns the day's events (failures, founder feedback,
surveys/research, successful patterns) into durable lessons. New rules land in `OW.md` (org-wide)
or a domain file (`ds.md`/`be.md`/`sf.md`/`mg.md`/`mb.md` — the per-team/agent memories). This
file is the running record of what was learned and whether mistakes repeat.

## Format (one section per UTC day)
```
## YYYY-MM-DD
- New lessons: OW-0XX (one line), DS-0XX (one line)...
- Repeat-mistakes: <count> — <rule code>: <what happened> (rule strengthened)
- Focus / areas improved: <one line>
```
