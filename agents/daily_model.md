## כללי ברזל — כל יום

1. **Deep work sacred** — בלוקי עבודה לא נשברים. אין exceptions.
2. **כל commit עובר JS validation** לפני push.
3. **חסם מדווח תוך שעתיים** — לא בסוף היום.
4. **כל פגישה → output מתועד** — החלטה / action item / scope חדש.
5. **Scope discipline** — כל סוכן עובד על הscope שלו בלבד.
6. **אפס hardcoded strings** — הכל דרך i18n file.
7. **אפס inline styles** ללא token reference מנטה.
8. **UI change** → token check + mobile viewport + RTL check.
