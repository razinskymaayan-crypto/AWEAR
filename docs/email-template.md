# תבנית סיכום דיון — AWEAR

כל סוכן שולח סיכום לכתובת החברה בסוף כל יום עבודה או בסיום דיון משמעותי.

---

## כותרת מייל

```
[AWEAR] [שם המחלקה] — סיכום [תאריך]
```

דוגמאות:
- `[AWEAR] Product — סיכום 16.06.2026`
- `[AWEAR] Backend — סיכום ספרינט שבוע 25`
- `[AWEAR] Board — סיכום פגישת הנהלה 16.06.2026`
