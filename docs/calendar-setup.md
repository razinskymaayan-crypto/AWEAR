# הגדרת יומן Google Calendar — AWEAR

## שלב 1 — פתיחת חשבון Google

1. פתחי חשבון Gmail חדש: `awear.team.il@gmail.com` (או שם אחר שתבחרי)
2. הוסיפי את כרמל ומעיין כמנהלות החשבון

---

## שלב 2 — יצירת לוחות שנה נפרדים

ב-Google Calendar יש ליצור לוח שנה נפרד לכל עובד/ה וסוג פגישה.
לכל לוח שנה — צבע שונה.

**איך יוצרים לוח שנה חדש:**
Settings (גלגל השיניים) ← Add calendar ← Create new calendar ← שם + צבע

| לוח שנה | צבע ב-Google Calendar | מי יוצר אירועים |
|---------|----------------------|-----------------|
| Board — כרמל + מעיין | Tomato (אדום) | כרמל / מעיין בלבד |
| כרמל | Grape (סגול) | כרמל |
| מעיין | Flamingo (ורוד) | מעיין |
| ג׳ף — CEO | Tangerine (כתום) | ג׳ף |
| איילון — Product | Banana (צהוב) | איילון |
| סטיב — CTO | Sage (ירוק בהיר) | סטיב |
| מארק — Dev | Basil (ירוק כהה) | מארק |
| וראן — Mobile | Peacock (טורקיז) | וראן |
| סאם — Backend | Blueberry (כחול) | סאם |

---

## שלב 3 — הרשאות

**כרמל + מעיין:**
- רואות את כל לוחות השנה
- יכולות לערוך את לוח ה-Board בלבד
- יכולות לצפות (לא לערוך) בלוחות של כל שאר הסוכנים

**כל שאר הסוכנים:**
- כותבים לאירועים בלוח השנה שלהם בלבד
- רואים את לוח ה-Board (קריאה בלבד)

**איך מגדירים שיתוף:**
פתחי את ההגדרות של כל לוח שנה ← Share with specific people ← הוסיפי את כתובת המייל של החברה ← "See all event details"

---

## שלב 4 — סוגי אירועים

| סוג פגישה | לוח שנה | כותרת מומלצת |
|-----------|---------|--------------|
| פגישת הנהלה שבועית | Board | `[BOARD] Weekly Sync — שבוע XX` |
| דיון מוצר | איילון | `[PRODUCT] [נושא]` |
| דיון טכני | סטיב | `[TECH] [נושא]` |
| דמו / pitch | Board | `[DEMO] [קהל] — [תאריך]` |
| One-on-one | לוח של הסוכן הרלוונטי | `[1:1] [שם] ↔ [שם]` |

## שלב 5 — חיבור האימייל ליומן

כשסוכן שולח סיכום דיון למייל החברה (`awear.team.il@gmail.com`), ניתן להוסיף את הפגישה ידנית ליומן מתוך המייל עצמו (Google מזהה תאריכים אוטומטית ומציע "Add to Calendar").

לחלופין — הסוכן יכול לשלוח invite ישירות ל-Google Calendar כחלק מהסיכום.
