# Market Research — autonomous competitive & UX research

When idle (no INBOX task, no open roadmap item), the autopilot alternates evenly between
**market research** (here) and **user surveys** (`docs/surveys/`).

Each research run picks a different topic tied to the app — a competitor (Whering, Depop,
LTK...), a UX pattern, onboarding, monetization, retention — and combines breadth (scan
several sources) with depth (study the most relevant closely). Findings + concrete
recommendations land here as one dated file; small obvious wins are implemented directly,
bigger ideas go to `.claude/master/IDEAS.md` for founder approval.

## File naming
`YYYY-MM-DD-<topic>.md`  (e.g. `2026-06-27-depop-resale-flow.md`)

All research is written in English.
